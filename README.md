# CustomScrollbar
TS custom scrolling solution

[Scrollbars demo](https://mchaov.github.io/CustomScrollbar/ "Scrollbars demo")

#### Desktop browsers support

* IE9 (requires ES5 and ES6 shims)
* IE: 10, 11, Edge
* Chrome
* Firefox
* Opera

#### Events support
* drag the scrollbar
* clikc on the track
* scroll with wheel
* scroll with keyboard:
..* cursors up/down
..* page up/down
..* home/end
